## Penjelasan Materi

### Control Statements
Control statements digunakan untuk mengatur alur eksekusi program, seperti percabangan (`if`, `else`, `switch`) dan pengulangan (`for`, `while`, `do-while`).

### List
List adalah struktur data yang digunakan untuk menyimpan sekumpulan data secara berurutan. List di Dart dapat berisi berbagai tipe data dan mendukung operasi seperti penambahan, penghapusan, dan iterasi.

### Looping
Looping digunakan untuk menjalankan kode secara berulang, misalnya dengan `for`, `while`, dan `do-while`. Looping sering digunakan untuk mengakses elemen dalam koleksi seperti List dan Map.

### Maps
Map adalah koleksi pasangan key-value. Setiap elemen Map memiliki kunci unik yang digunakan untuk mengakses nilainya. Map sangat berguna untuk menyimpan data yang terasosiasi.

### Sets
Set adalah koleksi data yang tidak memiliki duplikasi dan tidak berurutan. Set digunakan ketika ingin memastikan setiap elemen hanya muncul sekali dalam koleksi.
