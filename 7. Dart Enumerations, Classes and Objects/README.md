# Rangkuman Materi: Enumerations, Classes, dan Objects

## 1. Enumerations (Enums)
Enumeration atau Enum digunakan untuk mendefinisikan tipe data yang memiliki sekumpulan nilai tetap (konstan). Ini sangat berguna untuk mewakili pilihan yang terbatas, seperti jenis kelamin, status pesanan, atau arah mata angin. Menggunakan Enum membuat kode lebih aman karena mencegah penggunaan nilai yang tidak valid.

## 2. Classes dan Objects
**Class** adalah "cetak biru" (blueprint) atau template untuk membuat objek. Class mendefinisikan properti (variabel) dan perilaku (fungsi/metode) yang akan dimiliki oleh objek.

**Object** adalah perwujudan nyata dari sebuah Class. Jika Class adalah denah rumah, maka Object adalah rumah fisiknya.

## 3. Constructors
Constructor adalah fungsi khusus di dalam Class yang dijalankan secara otomatis saat sebuah objek dibuat. Gunanya adalah untuk memberikan nilai awal pada properti objek tersebut.

## 4. Methods (Fungsi di dalam Class)
Metode adalah fungsi yang didefinisikan di dalam Class untuk menentukan apa yang bisa dilakukan oleh objek tersebut (perilaku objek).
